## Features

- Random teleportation in Overworld & Nether & End
- Only teleport on surface & safe areas
- Adjustable maximum and minimum X and Z coordinates
- Circle & Square Algorithm
- Cooldown system
- Cancel on move
- Titles
- Highly customizable
- API
